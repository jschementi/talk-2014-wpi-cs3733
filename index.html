Fibonacci

def fib(n):
    if n == 0:
        return 0
    if n <= 2:
        return 1
        
    return fib(n - 1) + fib(n - 2)

O(2^n)

(technically O(1.618^n))

We can do much better

cache = {}

def fib(n):
    if n == 0:
        return 0
    if n <= 2:
        return 1
        
    value = cache.get(n)
    if value
        return value
        
    value = fib(n - 1) + fib(n - 2)
    cache[n] = value
    return value

O(n)

Call repeatedly, O(1)
